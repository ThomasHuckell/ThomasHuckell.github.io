Parameter Identification of CRS A465 Manipulator

Robotic manipulators require accurate dynamic model of the system for effective control system design, and for accurate planning, simulation and training. The models parameters such as link mass, inertia, and frictional properties are not always provided in technical manuals or accurate to the individual robot. Experimental identification is used to accurately estimate these values which consists of consists of robot modeling, trajectory generation, data acquisition, signal processing, parameter estimation and model validation. In the final project for ELEC 848 a graduate controls course, this process was carried out for the CRS A465 robot to build an accurate model.

Project Overview

We developed the dynamic model formulation using the Euler-Lagrange formulation. This expression is nonlinear in terms of the dynamic parameters, so it is reformulated into a reduced canonical form that is linear to model parameters.

With dynamics in this form one can appy least square regression to estimate these dynamic parameters.

A joint trajectory was generated such that all the identifiable parameters are continuously excited. With this trajectory we recorded three training data sets and processed the noisy joint acceleration with a gaussian filter. We performed least square regressions on all 3 data sets to estimate the dynamic parameters.

To validate the model we took two of the three training data sets took the average of the estimated parameters and used it to estimate the measured voltage of the third data set. We were able to predict the experimental results with an average Root Mean Square error for each joint of 0.088 Volts.
